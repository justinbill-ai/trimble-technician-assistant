How in-progress reports are saved

  1. Everything saves on this phone, tablet, or computer in the browser (not in the cloud). No account needed.
  2. Each machine gets its own named report — e.g. “336 DKS00000”. Notes and photos are stored with that name.
  3. Tap + New report to document another install. Your current report is saved first; nothing is overwritten.
  4. Come back later: use Quick open or the Saved reports list below to reload an unfinished report on this same device.
  5. Optional Backup this report exports a file you can import on another device. Delete this report removes it from this device only.
