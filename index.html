Technician Assistant

What is Technician Assistant?

A single repository for specialized Trimble field tools — organized by product category, each with enhanced documentation, checklists, and guided workflows. Categories organize tools on one page — open the tool you need directly. No app store install required.

  1. Open a tool — find it under its product category (Earthworks, Siteworks, TMC, and more) and click the tool row.
  2. Follow the guide — step-by-step instructions, reference photos, and exportable reports where supported.

Installation

End-to-end install documentation — capture machine wear and condition before Trimble machine control goes on, then photo and voice deliverables after the install for proof of quality and dealer reference.

How to use: Start with Pre-Inspection before the install (expert or guided wear mode, voice, camera, PDF). After install, open Install Deliverable — name each machine report, add unlimited photos and notes, switch between in-progress installs anytime, and export when finished. Dealer logo saves on this device for both tools.

Tools in this category

  • PI Pre-Inspection / Machine Wear Report Before install — expert or guided wear documentation with photos and Trimble-branded PDF ›
  • ID Install Deliverable After install — named reports, unlimited photos, voice notes, resume anytime, PDF export ›

How to use Technician Assistant

One home page for all specialized tools. Categories organize the list — you do not pick a category separately; just click the tool row under the product line you need. Each tool delivers enhanced documentation beyond a basic PDF or manual.
